# Biblias

Colección de Biblias en múltiples idiomas y formatos.

## Fuentes

Algunas Biblias de este repositorio han sido obtenidas del proyecto:

**[Holy-Bible-XML-Format](https://github.com/Beblia/Holy-Bible-XML-Format)** por [Beblia](https://github.com/Beblia)

## Estructura

Cada carpeta corresponde a un código de idioma (ISO 639) y contiene los archivos de la Biblia en ese idioma.
